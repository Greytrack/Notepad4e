Welcome to the Notepad4e update site!

It is recommended that you download and install the plugin via the Eclipse Marketplace, or simply drag and drop the following button to your running Eclipse workspace:

As an alternative, you can select Help -> Install New Software in the Eclipse IDE, paste the address of this webpage in the Work with: field of the window that opens up and press Enter on your keyboard. Select the Notepad4e plugin that appears in the list, click Next twice, accept the license and click Finish. Select OK or Install anyway when the warning about unsigned content appears. At this point the installation should have completed successfully, simply restart Eclipse and you're ready to go!
