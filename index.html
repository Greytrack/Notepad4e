Welcome to the Notepad4e update site!

It is recommended that you download and install the plugin via the Eclipse Marketplace.

As an alternative, in your Eclipse IDE you can select Help -> Install New Software -> Add... and enter the address of this webpage in the Location field. Click OK, select the Notepad4e plugin that appears in the list, click Next twice, accept the license and select OK when the warning about unsigned content appears. At this point the installation should have completed successully, simply restart Eclipse and you're ready to go!
